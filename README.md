YuezUrl
=======

**Version 0.0.1**

> `YuezUrl` is an url manipulator which can control url the current url in one
> broswer, is pure javascript and easily use.

**Usage**

    bower install YuezUrl

    yuezUrl = new YuezUrl
  
If you want to setting some params to the url, then

    yuezUrl.set {object}
